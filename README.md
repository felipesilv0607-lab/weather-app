# 🌤 Aplicativo do Tempo

Um aplicativo de previsão do tempo desenvolvido com HTML, CSS e JavaScript puro, consumindo a API do OpenWeather.

## 📸 Demonstração

*Em breve será adicionada uma imagem ou GIF do projeto.*

---

## 🚀 Funcionalidades

- Buscar o clima de qualquer cidade.
- Exibir temperatura atual.
- Mostrar sensação térmica.
- Exibir umidade do ar.
- Mostrar velocidade do vento.
- Exibir temperatura máxima e mínima.
- Mostrar descrição do clima.
- Exibir o ícone oficial do clima.
- Buscar utilizando o botão ou a tecla Enter.

## ⚙️ Como executar o projeto

1. Clone este repositório:

```bash
git clone https://github.com/felipesilv0607-lab/weather-app.git
```

2. Abra a pasta do projeto.

3. Crie uma conta em:
   https://openweathermap.org/

4. Gere sua API Key.

5. No arquivo `js/script.js`, substitua:

```javascript
const API_KEY = "SUA_API_KEY";
```

pela sua chave.

6. Abra o arquivo `index.html` utilizando a extensão **Live Server** do VS Code.
